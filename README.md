# SMM-ассистент со встроенным ИИ.

## Описание проекта
SMM-ассистента в виде удобного веб-приложения, способный генерировать посты и картинки, а также работать с API социальных сетей.
Веб-приложение, реализующее функционал генерации постов и изображений с последующей автопубликацией в ВКонтакте. 
Приложение развернутое на PythonAnywhere, поддерживает регистрацию и авторизацию пользователей.
